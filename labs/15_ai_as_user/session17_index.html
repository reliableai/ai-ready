Skillset 4 Building with AI and for AI

S17 — When AI is the User of Our System

Designing systems for AI consumers — what changes when the user is an agent, not a human: API ergonomics for agents, failure modes unique to agent consumers, and the shift from human-centred to agent-centred system design.

Learning objectives

  • Recognize how API and UX design shift when the consumer is an AI agent, not a human
  • Design tool schemas, error messages, and interfaces that agents can reliably use and recover from
  • Anticipate failure modes unique to agent consumers (prompt injection, tool misuse, runaway loops) and design for graceful degradation
