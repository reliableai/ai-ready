Sources of Uncertainty in AI Evaluation
Interactive labs for AI-Ready Engineers. Each playground lets you feel a source of uncertainty firsthand, then shows you how to fix it. Unified setup: an AI system scored 1–5 by an LLM judge on four metrics (Correctness, Helpfulness, Tone, Completeness).

Sampling Noise

Run the same evaluation twice — get different numbers. Set true scores, pick a sample size, and watch the scorecard wobble. Then tame it with beta posteriors, bootstrap CIs, and confidence intervals.

Model Accuracy Estimator

Enter successes out of n trials and get a posterior over accuracy: point estimate, mode, and 95% credible interval. Powered by the Beta distribution with selectable priors.

Sampling Bias

Your test set is 70% English but production is 40%. The scorecard looks great — until you see what users actually experience. Explore biased vs. representative sampling with adjustable language mixes.

interactive more data doesn't help #4

Multiple Hypothesis Testing

You test 10 prompt variants. The winner scores 4.3. Ship it? All variants had the same true score — the "winner" was just the luckiest. See how selection bias grows with N.

interactive selection bias #5

Variance Across Domains

Aggregate says 4.0. But Client A sees 4.5 and Client B sees 3.0. The aggregate hides the story. Includes a Simpson's Paradox demo where the "winning" system actually loses everywhere.

interactive property of the world #6

Choice of Metric

Mean says A wins. Threshold rate says B wins. Worst-case says B wins clearly. The winner changes with every tab click. The data didn't change — the question did.

interactive design decision #7

Temporal Drift

Your January eval says nothing about April. Simulate 12 months of gradual degradation and sudden shocks. See how long it takes to notice without continuous monitoring.

interactive fresh data over time #9

LLM Evolution

GPT-4: 97.6% → 2.4% in 3 months. Same prompt. Same task. The model changed under you. Real documented cases + simulated version timeline.

Judges & Compounding
