Course Project

Design, build, and evaluate an AI-powered system — and produce specific, defensible findings about how it behaves. Both tracks culminate in a presentation + discussion.

Tracks

  • Track A — Class System. Team project (1–5 participants). Pick one of the project directions presented in class. In-class reviews, working sessions, and design clinics throughout.
  • Track B — Capstone. Individual only. Work on one of the class projects individually, deliver at the exam.

Structure

Four sprints, plus one sprint to prepare the presentation. Each sprint produces concrete output (issues, commits, a short sprint note, a one-slide per-person update). Sprint-level specifics live in the project docs.

What every project ships

  1. Working code with a UI
  2. Project-specific data artifacts (versioned, replayable)
  3. Findings — specific, quantified observations about the system's behavior
  4. Technical report
  5. Final presentation + discussion

Methodology

Every project is expected to meet a baseline on a shared scaffolding (architecture, agent harness, memory/tools, logging, prompts as versioned files, resilience, cost, quality spec, smoke tests, and the behavior study). Details and minimum bars are in the universal scaffolding doc.

Grading (0–30, same rubric for both tracks)

Dimension (0–10 each) Focus
Engineering & System Working system + UI, clean module boundaries, versioned prompts, structured logging, reproducibility from a fresh clone.
Science & Findings Specific findings (not feature lists), quantified uncertainty, honesty about negative or surprising results, scope matched to team size and time.
Process & Communication Consistent per-sprint progress, individual accountability (per-person commits, defensible in-meeting), clear report and presentation with per-member contributions.

Project options

  • Conversational Clustering — cluster a dataset by talking to a human; the oracle is the sole judge of quality.
  • WhatsApp Personas — instantiate real-person personas as LLM agents and test individual and group fidelity.
